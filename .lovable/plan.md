

# AI-Powered Pitch Refinement Agent — Premium Landing Page

## Overview
A stunning, dark-themed SaaS landing page for an AI product that helps students refine their project pitches. Ultra-premium design inspired by Stripe/Notion with glassmorphism, glowing accents, and smooth animations.

## Color Theme
- **Background:** Deep dark navy (#0B1120 / #020617)
- **Gradient accents:** Indigo (#6366F1) → Cyan (#22D3EE)
- **Glow accent:** Soft purple (#A78BFA)
- **Text:** Light gray (#E5E7EB) on dark
- **Highlight:** Soft gold (#FBBF24) for special elements

## Sections

### 1. Floating Pill Navbar
- Rounded pill shape with backdrop blur (glassmorphism)
- Links: Home, About, How It Works, Features, Contact
- Highlighted "Get Started" CTA button with gradient
- Shrinks slightly on scroll with smooth transition

### 2. Hero Section
- **Headline:** "Turn Your Ideas into Winning Pitches"
- **Subtext:** AI-powered platform description
- Two buttons: "Try Now" (gradient) and "View Demo" (outlined)
- Decorative glowing orbs and abstract AI-themed visual elements
- Smooth fade-in entrance animation

### 3. About Section
- Clean two-column layout with icon blocks
- Explains the problem (weak pitch delivery) and the solution (AI virtual judge)
- Subtle fade-in-on-scroll animations

### 4. How It Works
- 5 animated step cards in a horizontal flow with connecting lines
- Steps: Submit Pitch → AI Evaluation → Score & Feedback → Improve Pitch → Track Progress
- Each card animates in on scroll with staggered timing
- Numbered steps with gradient icons

### 5. Features Section
- 6 premium feature cards in a 3×2 grid
- Features: AI Judge Simulation, Structured Scoring, Weakness Detection, Version Comparison, Pitch Success Predictor, Instant Feedback
- Glassmorphism card style with subtle borders
- Glow hover effect on each card
- Lucide icons for each feature

### 6. CTA Section
- Bold gradient background (indigo → cyan)
- **Text:** "Great Ideas Deserve Great Presentation"
- Glowing "Start Refining Now" button with hover animation

### 7. Footer
- Minimal dark footer
- Logo, quick links, contact info, social media icons
- Subtle separator line

## Animations & Effects
- Intersection Observer-based fade-in animations on scroll
- Parallax-style floating background orbs
- Glow hover effects on cards and buttons
- Smooth scroll between sections
- Navbar shrink transition on scroll
- Micro-interactions on buttons and interactive elements

## Technical Approach
- Single-page layout with anchor navigation
- Fully responsive (mobile, tablet, desktop)
- All animations via CSS + Tailwind utilities
- No backend needed — purely a landing page
